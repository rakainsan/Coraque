📡 Prediksi & Deteksi Anomali
POST /predict Prediksi volume + deteksi anomali + rekomendasi
Endpoint utama sistem. Menerima data 3 hari terakhir event sensor, mengembalikan prediksi SVR, hasil deteksi OCSVM, dan rekomendasi penanganan jika anomali terdeteksi.
Request Body (JSON)
FieldTypeStatusKeterangan
events array[float] Required Array 3 nilai event sensor: [t-3, t-2, t-1] — urutan paling lama ke terbaru
jumlah_kain float Required Jumlah kain yang diproduksi hari ini (lembar)
panjang_kain_m float Required Rata-rata panjang kain per lembar (meter)
source string Optional Sumber request untuk logging. Default: "-"
Contoh Request
{ "events": [22, 24, 25], // [t-3, t-2, t-1] "jumlah_kain": 120, "panjang_kain_m": 2.7, "source": "IoT CORAQ" }
Response 200 — Kondisi Normal
{ "svr": { "forecast": 25.3241 // prediksi jumlah event besok }, "ocsvm": { "is_anomaly": false, "label": 1, // 1=normal, -1=anomali "score": 0.0443 // positif=normal, negatif=anomali }, "rekomendasi": { "tipe": "normal", "severity": "normal", "emoji": "✅", "tindakan": "Kondisi volume limbah normal. Tidak diperlukan tindakan khusus.", "bahan_kimia": [], "infrastruktur": [] }, "telegram_sent": false }
Response 200 — Anomali Terdeteksi
{ "svr": { "forecast": 26.1823 }, "ocsvm": { "is_anomaly": true, "label": -1, "score": -0.0089 }, "rekomendasi": { "tipe": "spike", "severity": "sedang", "emoji": "🚨", "tindakan": "Volume limbah signifikan di atas normal...", "bahan_kimia": [ { "nama": "PAC (Poly Aluminium Chloride)", "dosis": "50 mg/L", "lokasi": "Bak 3 — Sedimentasi", "alasan": "PAC membentuk flok lebih cepat..." } ], "infrastruktur": [] }, "telegram_sent": true }
📈 Prediksi 7 Hari
POST /predict/rolling Forecast rolling SVR hingga 7 hari ke depan
Request Body (JSON)
FieldTypeStatusKeterangan
events array[float] Required Minimal 3 nilai event sensor terbaru
days integer Optional Jumlah hari forecast. Default: 7, maksimal: 7
Contoh Request
{ "events": [22, 24, 25], "days": 7 }
Response 200
{ "events": [22, 24, 25], "days": 7, "forecasts": [ { "day": 1, "forecast_event": 25.3241, "forecast_liter": 7394.64, // forecast_event × 292 "upper_event": 26.5903, "lower_event": 24.0579, "upper_liter": 7764.37, "lower_liter": 7024.91 } // ... hari 2–7 ] }

⚖️ Severity Levels
Severity ditentukan dari OCSVM decision function score secara langsung — bukan threshold arbitrary. Threshold berbasis distribusi empiris data training.
SeverityOC ScoreTipeTindakan Utama
normal ≥ 0 — Tidak ada tindakan
ringan -0.0062 s/d 0 Spike Tambah Tawas + PAC 10–15%
sedang -0.0123 s/d -0.0062 Spike PAC 50 mg/L + Tawas 150 mg/L
berat < -0.0123 Spike PAC 75–100 mg/L + Tawas + Bak Ekualisasi
peringatan < 0 + drop Drop Inspeksi sensor & sistem

🐘 Contoh Integrasi Laravel
Controller Laravel — Memanggil /predict
// PredictionController.php use Illuminate\Support\Facades\Http; public function predict(Request $request) { $response = Http::post('http://202.155.90.7:5000/predict', [ 'events' => [$request->lag3, $request->lag2, $request->lag1], 'jumlah_kain' => $request->jumlah_kain, 'panjang_kain_m' => $request->panjang_kain_m, 'source' => 'Laravel CORAQ', ]); $data = $response->json(); // Simpan ke database Prediction::create([ 'svr_forecast' => $data['svr']['forecast'], 'is_anomaly' => $data['ocsvm']['is_anomaly'], 'ocsvm_score' => $data['ocsvm']['score'], 'severity' => $data['rekomendasi']['severity'], 'rekomendasi' => json_encode($data['rekomendasi']), ]); return response()->json($data); }
